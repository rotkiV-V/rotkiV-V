Szövegszerkesztők

Notepad++

A Notepad++ egy szöveg- és forráskódszerkesztő Windows platformra, de Linuxon és Macen is futtatható Wine segítségével.

TextMate

A TextMate egy általános célú GUI szövegszerkesztő a macOS rendszerhez, amelyet Allan Odgaard készített

Atom

Egy népszerű szövegszerkesztő, amely sok pluginnal rendelkezik, rengeteg testre szabható dolog található benne a felhasználó igényeinek megfelelően.

Visual Studio Code

Egy nagyon funkcionális és erőteljes szövegszerkesztő, amelyet széles körben használnak a webfejlesztők. Támogatja a kódkiegészítést, a hibajavítást és az integrált verziókezelőt.
